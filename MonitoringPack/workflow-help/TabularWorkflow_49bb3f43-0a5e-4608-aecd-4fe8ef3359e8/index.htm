Monitor Volumes and Aggregates

 

Goal

This workflow will Monitor all the volumes and aggregates whose used capacity is greater than or equal to the threshold values provided.The monitored results will be mailed to the specific user specified in user inputs.
This work flow has following comands

·         Monitor Volumes: gives list of all volumes whose used percentage is greater than provided volume threshold.

·         Monitor Aggregates: gives list of all volumes whose used percentage is greater than provided Aggregate threshold.

·         Monitor Volumes and Aggregates: gives list of all volumes and aggregates whose used percentage is greater than provided volume_threshold and aggregate threshold.

·         Convert CSV to HTML: Monitoring commands generate reports in CSV format and this command converts CSV to HTML for easy readability

·         Send Email Reports:This command emails the genrerated reports to a specific user.

·         Scheduler:This command schedules the work flow for recurring execution.

User Inputs

Threshold Input parameters:

1) Aggr_Threshold(required) – This takes input range from 1-100 and filters all the aggregates whose used percentage greater the given input..

2) Volume_Threshold(required) – This takes input range from 1-100 and filters all the volumes whose used percentage greater the given input..


Schedule Input parameters:

Schedule Inputs(atleast one required) – This takes several interval parameters like day,hour,minute,year,months and week at which the workflow should execute.Atleast one parameter is mandatory.By default day is set to 1.


E-mail Input parameters:

Email Inputs(all required) – This takes inputs like To address,From address and SMTP IP/Hostname.It attaches the reports and sents the email to a specific user mentioned in To address.


 

Abort

This workflow will execute recursively according to the schedule.If you want to abort this workflow from execution you can do it by going to execution tab,click on execution status and then right click and choose abort.


Result

The workflow will monitor volumes and aggregates above threshold and send mail by attaching the reports to the user.
The work flow will send reports at specified intervals continiously.
